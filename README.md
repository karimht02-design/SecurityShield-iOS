# 🛡️ SecurityShield - تطبيق الحماية المتقدم للآيفون

تطبيق iOS شامل لحماية الجهاز والبيانات الشخصية بميزات متقدمة جداً.

## ✨ الميزات الرئيسية

- 🔒 **تشفير البيانات الكامل** - حماية البيانات الحساسة
- 🕵️ **كشف التهديدات الفورية** - اكتشاف المخترقين والاتصالات المريبة
- 📍 **VPN مدمج** - إخفاء الموقع الحقيقي وتغييره
- 🚨 **نظام التنبيهات** - إرسال تنبيهات فورية عند اكتشاف خطر
- 🔐 **حماية كلمات المرور** - تخزين آمن للبيانات الحساسة
- 🛡️ **جدار ناري ذكي** - حجب الاتصالات والعناوين الضارة
- 📊 **لوحة تحكم متقدمة** - مراقبة شاملة وإحصائيات مفصلة
- 🎯 **فحص تلقائي** - كشف مستمر للتهديدات

## 🚀 البدء السريع

### المتطلبات:
- iOS 14.0+
- Xcode 13+
- Swift 5.5+

### التثبيت:
```bash
git clone https://github.com/karimht02-design/SecurityShield-iOS.git
cd SecurityShield-iOS
open SecurityShield.xcodeproj
```

ثم اضغط ▶️ Run في Xcode

## 🛠️ التطوير

### الملفات الرئيسية:

- `SecurityShieldApp.swift` - نقطة الدخول
- `SecurityService.swift` - خدمة الأمان الرئيسية
- `ThreatDetector.swift` - كاشف التهديدات
- `Models/ThreatModel.swift` - نماذج البيانات
